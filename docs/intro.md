# 기술 블로그 작성시작
======================================================
## &#91;KDT&#93; 기술 블로그 작성

**기술 지식**의 정리, **질문 작성**을 통한 복습, 글쓰기를 통한 **커뮤니케이션 능력 향상**

## 작성 요령

**필수 항목**
- 온라인콘텐츠 학습 범위
- 학습한 내용 요약본
- 퀴즈 출제
    - 3문제 이상
- 출결 확인용 사진,캡처본

**기타**
- 실패와 해결에 관한 기술
- 실험과 결과에 관한 기술
