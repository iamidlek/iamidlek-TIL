#### 문제 3. .arrow-grow는 어디가 기준이 될까?
```css title="html 구조를 따르고 있다"
.signboard__items .shopnow__items a {
  position: relative;
}
.border-move{
  position: absolute;
}
a:hover .border-move{
  width: 92%;
  height: 70%;
}
.shopnow__items a h3{
  font-size: 16px;
  font-weight: 500;
  letter-spacing: 2px;
  padding-left: 22px;
}
.arrow-grow{
  position: absolute;
}
.arrow-body{
  border-top: 2px solid transparent;
  position: absolute;
}
```
1. .border-move
2. a
3. .arrow-body 
4. h3
