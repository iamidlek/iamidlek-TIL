## Quiz
#### 문제 1. 1970년대 UNIX에 구현된 기술이 아닌것은?
1. 멀티 태스킹
2. 시분할 시스탬
3. 멀티 프로세싱
4. 멀티 프로그래밍

#### 문제 2. 멀티 프로그래밍의 설명이 아닌것은?
1. 응용프로그램은 CPU를 온전히 많이 사용한다
2. 응답시간을 짧게한다
3. 다중 사용자를 지원한다
4. CPU를 최대한 가동하게 한다


#### 문제 3. 선점형 스케쥴러의 설명이 아닌 것은?
1. 알고리즘 종류로 FCFS 가 있다
2. 알고리즘 종류로 SJF 가 있다
3. 알고리즘 종류로 Round Robin 이 있다
4. 프로세스의 자발적 blocking 상태를 기다린다
