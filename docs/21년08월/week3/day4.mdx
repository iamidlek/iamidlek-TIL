## Quiz
#### 문제 1. 자식요소 &lt;a&gt; 를 중앙정렬하기위한 조합이 아닌것은?

```css
div{
  width: 300px;
  height: 300px;
}

1. div{
  display: flex; 
  justify-content: center;
  align-items: center;
  }

2. a{
  display: block; 
  width: 5px;
  height: 5px;
  margin: auto; 
  } 

3. div{
  position:relative
  }
  a{
  position:absolute;
  width: 50px;
  height: 50px; 
  margin-top:50%;
  top: -25px;
  margin-left:50%;
  left: -25px
  }
  
4. div{
  width: 300px;
  height: 300px;
  text-align: center;
  line-height: 300px ;
  }
```
<details>
  <summary>
  답안 확인
  </summary>
  <ul class='quiz'>
    <li>②</li>
    <li>2번의 경우 좌우 정렬만 일어난다</li>
    <li>1번의 경우 자식이 인라인이나 블록이나 가능</li>
    <li>3의 경우엔 요소 자체는 중앙 정렬된다 (a 내부 별개)</li>
    <li>4번의 경우 인라인요소를 유지하면서 정렬이 된다</li>
  </ul>
</details>
