## Summary
#### 인터럽트
- CPU가 프로그램을 실행하고 있을 때,  
입출력 하드웨어 등의 장치나 또는 예외상황이 발생하여  
처리가 필요할 경우에 CPU에 알려서 처리하는 기술
- 인터럽트의 종류  
  - 0으로 나누는 경우
  - 타이머 인터럽트
  - 입출력 인터럽트

#### 프로세스와 컨텍스트 스위칭
- 프로세스의 구성
  - 코드
  - 데이터
  - 스택
  - 힙
- PCB
  - 프로세스 ID
  - 레지스터 값(PC, SP)
  - 스케쥴러
  - 메모리 정보

#### IPC(InterProcess Communication)
- 프로세스간 통신방법
  - file 사용
  - 메세지 큐
  - 메모리 공유
  - Pipe
  - Signal
  - Semaphore
  - Socket

- 대부분의 IPC 기법은 커널을 통해 이루어진다
