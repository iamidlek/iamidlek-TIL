## Quiz
#### 문제 1. nvm과 npm의 설명으로 틀린 것은?
1. nvm → Node.js → npm 순의 설치가 가능하다
2. nvm도 버전 업그레이드가 가능하다.
3. npm 은 라이브러리와 패키지를 편리하게 설치하게 해준다.
4. npm으로 설치한 node 모듈들은 지우면 큰 문제가 된다.
